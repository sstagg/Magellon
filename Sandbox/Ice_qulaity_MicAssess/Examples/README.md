# Example star files

| File | Description |
|------|-------------|
| `minimal.star` | Simplest valid input — one `data_` block, `_rlnMicrographName` only |
| `relion31.star` | RELION 3.1 format with optics-group block; output star files preserve this structure |

Both files use placeholder paths (`micrographs/session_00N.mrc`).
Replace those paths with your real `.mrc` file paths before running.
