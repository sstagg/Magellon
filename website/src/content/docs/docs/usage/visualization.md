---
title: Data Visualization
description: Explore imported sessions in the Magellon UI.
---

Once a session has imported, open **Images** to browse the micrographs. From there you can:

- Filter by magnification, defocus, or any annotated metadata.
- Open a micrograph for full-resolution viewing with overlay toggles.
- Inspect per-image job history — CTF fits, motion-correction results, particle picks all attach to the underlying image.
- Launch a plugin run against the currently selected image or the whole session from the **Plugins** page.

Plugin results render inline with plugin-specific views — Thon-ring sketches for CTF fits, drift trajectories for motion correction, and particle overlays for pickers. The raw outputs remain on disk under the job's output directory.
