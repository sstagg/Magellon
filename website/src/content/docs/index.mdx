## What Magellon does

<CardGrid stagger>
    <Card title="Plugin platform" icon="puzzle">
        Drop a Python plugin with Pydantic schemas; the UI renders forms automatically.
        CTF estimation, motion correction, particle picking — all under one roof.
    </Card>
    <Card title="Batch jobs" icon="list-format">
        Submit work on a single image or a whole session. Progress streams live over Socket.IO,
        and every result is persisted to the database.
    </Card>
    <Card title="Per-plugin result views" icon="seti:plan">
        Thon-ring sketches for CTF fits, drift trajectories for motion correction, particle
        overlays for pickers — tailored visualisations, not raw JSON.
    </Card>
    <Card title="Docker-first" icon="seti:docker">
        `bash start.sh` and the full stack is up. The same containers run on a laptop and a cluster.
    </Card>
</CardGrid>
